## 📌 Description

A smart lead capture plugin for WordPress that enables custom form creation and shortcode integration, while supporting test emails, SMTP configuration, and WordPress default wp_mail() functionality. It also includes custom email template support directly from the admin to enhance engagement and conversions, along with GDPR-compliant unsubscribe functionality.

Designed for marketers and developers, this plugin helps streamline lead collection and automate personalized communication workflows without relying on third-party limitations.

---

## ✨ Features

### 🧩 Custom Lead Forms
Create and manage lead capture forms with flexible fields tailored to your business needs.

### 🔗 Shortcode Support
Easily embed forms anywhere on your website using simple shortcodes.

### 📧 Custom HTML Email Templates
Design and send fully customized HTML email templates directly from the WordPress admin panel.

### ⚡ Email Automation
Trigger automated emails upon form submission with dynamic content.

### 🔒 GDPR-Compliant Unsubscribe
Built-in unsubscribe functionality to ensure compliance with GDPR and email regulations.

### 🚀 Secure & Optimized
Includes nonce validation and follows WordPress coding standards for security and performance.

---

## ⚙️ How It Works

1. Install and activate the plugin  
2. Create your lead form from the admin panel  
3. Embed the form using shortcode  
4. Configure your email template  
5. Capture leads and automate responses instantly
